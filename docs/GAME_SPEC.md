# Survivor Stage ゲーム仕様書（v1.1）

固定カメラのステージ制サバイバー。Phaser 3 + TypeScript + Vite。

---

## 基本

| 項目 | 値 |
|------|-----|
| 解像度 | 960 × 540 |
| ステージ数 | エリアごと（Plains 3 / Forest 5） |
| エリア選択 | タイトルで選択。Plains 最初から、Forest は Plains クリア後 |
| 1ステージ時間 | 全ステージ 30秒（残り10秒で FINAL WAVE） |
| カメラ | 固定（スクロールなし） |
| 床 | Plains / Forest はタイルシート（縦5色を上から順に Stage 1→）を敷き詰め。Forest はさらに枝・草・葉の装飾をランダム配置。他エリアは Rectangle の単色 |
| UI | Rectangle + Text のみ（DOM 不可） |
| 物理 | arcade のみ（`fps: 240 = 60×4サブステップ`, `fixedStep: true`） |

**物理方針（Phaser 任せ）**: 移動・弾・当たり判定はすべて Arcade Physics に任せる。`body.setVelocity` で動かし、`physics.add.overlap` で衝突を検出する。旧方式（座標直接更新 + 距離計算）は `docs/ROLLBACK_MANUAL_PHYSICS.md` を参照。

**高速弾のすり抜け対策**: Arcade Physics には CCD（連続衝突判定）がないため、物理の固定ステップを 1/240 秒（1描画フレームを4分割）に細分化する。弾速 800px/s でも1ステップの移動は約 3.3px となり、判定を飛び越えない。速度は px/秒 指定なので体感速度は変わらない。当たり判定サイズ自体は拡大しない（弾の判定は半径 4px のまま）。

### 操作

- WASD / 矢印キー優先
- キー未入力時のみポインタ／タッチ追従
- 斜め移動は normalize して速度一定

### 成長ループ

敵撃破 → コイン → 拾うと XP → レベルアップ 3択（ステージ中のみ）

ステージクリア時は永続ゴールドを獲得し、画面中央から上部バーのゴールド欄へ
金色のキラキラが飛ぶ。ゴールドはゲームオーバーでも減らず、Clear Save でのみ消える。

### ステージクリア・ゴールド

- Plains: 通常 1G / 最終 2G
- Forest: 通常 2G / 最終 4G
- 以降のエリアも順番に基礎額が +1G（Volcano 3G、Ruins 4G…）
- そのステージをノーダメージかつ時間前に全敵撃破すると、上記報酬を2倍
- ゴールドは獲得演出開始時に即保存する
- ゴールドはタイトル画面のショップで永続強化に使う

### ショップ（実装済み）

- タイトル下部の Shop 枠をクリック、または選択して SPACE / ENTER で開く
- 2列の商品カード（最大3段）。マウスまたは WASD / 矢印で選択、SPACE / ENTERで購入、ESCで戻る
- 初回価格は各1G。購入するたびに同じ商品の価格が +1G（1、2、3…）
- Max HP: すべての新規ランの開始最大HP +1
- Power Cap / Speed Cap / Range Cap: 対応するラン中レベル上限 +1
- Power / Speed / Range の初期上限は3
- Pierce / Blast の初期上限は1。各スキルを実績で入手すると、対応する Cap 商品がショップに追加される
- Pierce Cap / Blast Cap: 対応するラン中レベル上限 +1
- XP Bonus の初期上限は2。Plainsクリア後、XP Bonus Capがショップに追加される
- Seal Slot: レベルアップ候補から封印できるスキル数 +1。初期枠は0
- Seal Slot価格は10G、20G、30G、40G、50G…と購入ごとに10Gずつ上昇
- 購入内容と残りゴールドは即時保存し、Clear Saveでリセット

### スキル封印

- タイトル画面のShop横にある `Seal Skills` から設定する
- Power / Speed / Range と解放済みスキルを、購入済み封印枠の数まで封印できる
- 封印したスキルはレベルアップ候補に出ない。解除すれば再び候補に戻る
- Goldは封印対象外。封印状態はセーブへ永続保存する

### 死亡

敗北画面の **TITLE** でタイトルへ戻る（その場での Stage 1 リトライはしない）。  
タイトルに戻ったあとは、エリアを選ぶと常に **Stage 1** から始める（途中再開なし）。

### ステージ進行

ステージクリアで次へ進むときは **レベル・XP・強化ステータスを引き継ぐ**（同じプレイ中のみ）。HP は次ステージ開始時に全回復。  
途中でタイトルに戻った場合、強化の引き継ぎは残らない。

### エリア選択（タイトル）

タイトルでエリアを選んでから開始する。

| エリア | 英語名 | ステージ数 | 状態 |
|--------|--------|------------|------|
| 平原 | Plains | 3 | ✅ 最初から選択・開始可。Stage 3 が最終（強化難易度） |
| 森 | Forest | 5 | Plains クリアで解除。Stage 5 が最終 |
| 火山 | Volcano | 5 | Forest クリアで解除 |
| 遺跡 | Ruins | 5 | Volcano クリアで解除 |
| 城 | Castle | 5 | Ruins解放まで「?」。Ruinsクリアで解除 |
| 地下迷宮 | Dungeon | 5 | Castle解放まで「?」。Castleクリアで解除 |

操作: **W/S**（または↑↓）で選択、**SPACE / ENTER** で決定。  
LOCKED も選択できるが決定はできない。選択中に解除条件を表示する。

### Volcano の敵

- Stage 1: 通常スライムのみ（エリア開始時の難易度を抑える）
- Stage 2: 装甲敵。1発3ダメージ未満を無効化し、Power 強化を促す
- Stage 3: 盾敵。プレイヤー側の正面攻撃を防ぎ、側面・背後からの跳弾だけ通す
- Stage 4: 盾敵。通常敵も混ざって出現する
- Stage 5: 高速・突進・装甲・盾・射撃敵が混ざり、固まらずランダム位置へ連続出現する。高速敵は初期の通常弾2発で倒せ、プレイヤーより少しだけ速い

必要な Move・Power・Ricochet が不足している場合は、レベルアップの3択へ必ず含める。

### セーブ（実装済み）

**保存先**: `localStorage`（キー `survivor-stage-unlocks`）。F5 更新後も残る。Boot では消さない。

**保存内容**:
- 実績・スキル解放・エリアクリア
- 生涯統計（試行回数・撃破数など）
- 途中ランの再開データは **保存しない**

**タイトル UI**:
| 要素 | 説明 |
|------|------|
| Clear Save | 設定メニューのみ。確認ダイアログ（初期 **NO** / **Yes clear data**） |
| 設定（⚙） | 右上。右からメニュー。BGM ON/OFF / Clear Save（タイトル）/ Give Up to Title（ゲーム・要確認） |

パスワード／引き継ぎコードは使わない（同じ端末の自動セーブのみ）。

**その他**:
- エリアを選んで開始 → 常に Stage 1 から
- エリア最終クリア（GAME CLEAR）→ 解放は残る
- 死亡・Give Up → タイトルへ戻る（次回は Stage 1 から）
- WASD で遊んでいるとき、次ステージでもキーボード移動を維持（クリックでマウス追従に戻る）

## クリア条件（ステージ）

1. **主条件**: 制限時間経過 かつ HP > 0
2. **副条件（速攻）**: 全ウェーブ終了後、敵 0 体 → 即クリア

制限時間前に全敵を倒した場合は、STAGE CLEAR のあとに
**ALL ENEMIES CLEAR!** を表示し、中央からXP欄へキラキラを飛ばして **XP +5**。
バナー直後に実報酬を大きな文字でトゥイーン表示する。通常は `+5 XP`、
そのステージをノーダメージで全敵撃破した場合は
`NO DAMAGE · GOLD ×2` も追加し、その後にXP・ゴールドのキラキラを各HUD欄へ飛ばす。
XP BonusがLv2～3ならこのクリアXPは2倍、Lv4～5なら3倍（以降も2レベルごとに+1倍）。

### ステージ結果 UI（Day 6 実装済み）

| 結果 | 表示 | ボタン |
|------|------|--------|
| ステージクリア | STAGE CLEAR! | NEXT STAGE → 次ステージへ |
| 全ステージクリア | GAME CLEAR! | TITLE → タイトルへ |
| 死亡 | DEFEATED | TITLE → タイトルへ戻る |

死亡時・クリア時は時間停止。クリア後の次ステージは成長を引き継ぎ、タイトルからの新規開始だけリセット。

### ステージクリア後の順番（固定）

1. 大きな STAGE CLEAR! 文字（tween）＋戦闘 BGM 停止→クリア BGM
2. 落ちているコインを全画面からプレイヤーへ吸引（XP 加算）
3. たまったレベルアップをすべて選択
4. 四角パネルの STAGE CLEAR / GAME CLEAR 表示

クリア BGM ファイル: `public/assets/audio/bgm_clear.ogg`（未配置時は無音で続行）

### 難易度（レベルアップ成長に連動）

想定: 1ステージあたり約 1.25 回レベルアップ、火力はレベル1回あたり約 +35%。

- 敵 HP ≈ 基本2 × 想定プレイヤー火力
- 敵速度はステージごとに約 +7%
- 出現数はステージが進むほどバースト人数が増える（30秒・22秒までスポーン）
- Plains Stage 2 の敵HPは敵ごとに3または4（各50%）とし、Power強化による必要攻撃回数の変化を出す
- Plains Stage 3 は最初の最終面なので、敵数・HP・速度・射撃敵率の最終面補正を弱める
- Plains Stage 3 の射撃敵は、距離を取るぶん近接敵よりHPを25%低くする

## 爽快感・フィードバック設計（v1.1 追加）

仕組みを大きく変えず、テンポと手応えを上げるための設計。

### 方針

- ゲームの正体は **部屋クリア型ステージ制**（全面 VS クローンにはしない）
- サバイバー系から借りるのは **密度・成長・フィードバック** のみ
- グラフィック資産がなくても、**音 + Primitive 演出** で爽快感の 7 割を担う

### コイン吸引（実装済み）

| 項目 | 値 |
|------|-----|
| 吸引開始半径 | 80px（プレイヤー中心） |
| 初速 | 120 |
| 最高速度 | 520 |
| 加速度 | 2200 / 秒² |
| 移動方式 | `body.setVelocity` + `fixedStep: true`（60fps） |

**意図**: コインを拾いに寄るストレスを減らし、戦闘の流れを切らない。

### 最小効果音セット（実装済み）

| キー | タイミング | イメージ |
|------|-----------|----------|
| `sfx-enemy-defeat` | 敵撃破時 | 短い下降音 |
| `sfx-player-fire` | プレイヤー弾発射時 | `player_fire.ogg`（Launch.wavを変換） |
| `sfx-coin-pickup` | コイン取得時 | 短い高音 |
| `sfx-player-hurt` | プレイヤー被ダメ時 | 低い衝撃音 |
| `sfx-level-up` | レベルアップ時 | 上昇音 |
| `bgm` | ゲーム中 | `bgm_loop.ogg` をループ再生 |

- 発射音だけ外部OGGを使い、それ以外の効果音は Web Audio で生成
- BGM は `public/assets/audio/bgm_loop.ogg`

### 演出（Day 5 で実装）

| 優先度 | 内容 | 状態 |
|--------|------|------|
| 高 | レベルアップ 3択 + 停止 + 効果音 | ✅ |
| 高 | 撃破時の白フラッシュ（0.1秒） | ✅ |
| 中 | 被弾時の画面フラッシュ | ✅ |
| 低 | 撃破時の極短ヒットストップ（0.05秒） | 未（調整フェーズ） |

### 数値チューニング方針

- 序盤 30 秒で「強くなった」を体感させる（レベルアップ実装後）
- 攻撃間隔の初期値は 800ms。発射速度レベルが 1,2,3… と上がると間隔は 800/レベル になる
- 敵の同時出現数は段階的に増やす（Stage 1 は開発用に調整中）

| HP | 基本2 × (火力伸びの約半分)。以前より軟らかく、貫通で群れを処理しやすい |

### Forest 後半の特殊敵

- Stage 3以降は、通常敵・射撃敵に加えて装甲敵と突進敵が出現する
- 装甲敵: 1発3ダメージ以上でなければHPが減らない。HPは同ステージの通常敵の約1.25倍
- 突進敵: プレイヤーとの距離が150px以内になると、その瞬間の方向へ直線突進する。突進中は方向を変えない
- 特殊敵の合計出現率は Stage 3=40%、Stage 4=50%、Stage 5=60%。装甲敵と突進敵は同率

### 射撃型（Day 7 実装済み）

| 項目 | 値 |
|------|-----|
| 出現開始 | Stage 3 |
| 出現割合 | Stage 3–4: 約25% / 5–7: 約45% / 8–10: 約60% |
| 移動速度 | 近接の 0.55 倍 |
| 好みの距離 | プレイヤー攻撃レンジの外側（レンジ +24px 〜 さらに +56px の帯）。レンジが広がると立ち位置も外へ。プレイエリア内に留まる |
| 射撃条件 | プレイヤー攻撃レンジの外にいるときだけ撃つ |
| 射撃間隔 | 1400ms |
| 敵弾 | 速度 280・ダメージ 1・画面上最大 15・`overlap` で被ダメ |

## レベルアップ 3択（実装済み）

タイトル新規開始／死亡リトライ時はリセット。ステージクリア時は次へ持ち越し。スタック可。  
毎回、候補プールから **ランダムに3つ** を出す。

次のレベルに必要なXPは **4、7、11、16、22…**。
必要量の増加幅を3、4、5、6…と1ずつ増やし、必要XPが **50** に達した後は
それ以上増やさず、以降も毎回50XPとする。

| 選択肢 | 効果 | 状態 | 初期ロック |
|--------|------|------|------------|
| Power | Fire Damage +1 | ✅ | なし（常に候補） |
| Speed | Fire Speed +1（1→2→3…。間隔 = 基本間隔 / 速度） | ✅ | なし（常に候補） |
| Range | Fire Range +1（射程 ×1.25） | ✅ | なし（常に候補） |
| Move | プレイヤー移動速度 +24 | ✅ | **Forestクリアで解放（暫定）** |
| Pickup | コイン吸引半径 +28px（表示名。内部IDは magnet） | ✅ | **Forestクリアで解放（暫定）** |
| XP Bonus | コインは常に1枚=1XP。Lv1=50%で2枚、Lv2=常時2枚、Lv3=50%で3枚（それ以外2枚）、Lv4=常時3枚。以降も同じ規則 | ✅ | **Plainsクリアで解放** |
| Pierce | Hit +1 more enemy（未取得=1体、1回目=2体…）。同一敵には1回だけ | ✅ | **実績で解放するまで候補に出ない** |
| Blast | Damage nearby on hit。取るほど円が大きい | ✅ | **実績で解放するまで候補に出ない** |
| Ricochet | 命中後、未命中の最寄り敵へ向きを変える。取得ごとに跳弾 +1 | ✅ | **Forestクリアまで候補に出ない** |
| 撃破爆発 | （旧案・撃破時）→ 命中時爆破に変更済み | — | — |

- 初期状態の候補は Power・Speed・Range の3つだけ
- 未解放スキルは候補プールから除く
- HPはレベルアップ候補に出さず、ショップの永続強化だけで増やす
- Power・Speed・Range・Pierce・Blastはショップで購入した現在の上限に達すると候補から除く
- 全スキルが上限で候補が0件になった場合だけ、`Gold — Gain 1 Gold` を1枚表示する
- 通常の強化候補が1つでも残っている間、Gold はレベルアップ候補に出さない

レベルアップ中: 物理・タイマー停止 + 3択 UI  
レベルアップ時: 通常エリアは HP 全回復。**Ruinsは全回復しない**

### 実績ロック解放（第1弾・実装済み）

目的: 実績を取るほど「できること」が増える。

- **ゲームクリア時**に条件を判定（途中ステージクリアでは解放しない）
- 達成すると称号（実績）が付き、対応スキルが候補プールに入る
- `localStorage` キー `survivor-stage-unlocks` に解放済み実績 ID を保存（ブラウザを閉じても維持）
- 死亡リトライ・ステージ持ち越しでは消えない
- 被ダメ／貫通・爆破取得はラン全体で記録し、ステージ跨ぎでも持ち越す

| 称号 ID | 表示名 | 条件 | 解放スキル | 状態 |
|---------|--------|------|------------|------|
| `untouched` | Unlock Pierce | Clear Plains with no damage | Pierce | ✅ |
| `pure_power` | Unlock Blast | Clear Plains without upgrading Power | Blast | ✅ |
| `forest_clear` | Unlock Move | Clear Forest（暫定） | Move | ✅ |
| `forest_clear` | Unlock Pickup | Clear Forest（暫定） | Pickup | ✅ |
| `forest_clear` | Unlock Ricochet | Clear Forest（暫定） | Ricochet | ✅ |
| `plains_clear` | Unlock XP Bonus | Clear Plains | XP Bonus | ✅ |
| `forest_clear` | Unlock Volcano | Clear Forest | Volcano | ✅ |

表示名は「達成すると何が解放されるか」が分かる名前にする（実績パネルはバトルHUDより手前の不透明全画面＋囲み枠）。

クリア結果画面に新規解除があれば短い文言（例: `UNLOCKED: Pierce`）を表示する。

### 貫通（確定仕様）

- `pierceLevel` 初期 0。選ぶたびに +1
- 弾の最大ヒット数 = `pierceLevel + 1`
  - 未取得（0）→ **1体**で消滅
  - 1回取得 → **2体目**で消滅
  - 2回取得 → **3体目**で消滅
- 当たった敵 UID を記録し、同じ弾では再ヒットしない
- processCallback でヒット上限・再ヒットを事前に弾く

### 範囲爆破（命中時）

- `blastLevel` 初期 0。選ぶたびに +1
- 周囲ダメージ: 1回目=1、2回目=2…（上限は弾の本来ダメージ）
- 半径: 1回目≈28px、以降 +12px
- 弾が敵に当たったときに発動（撃破不要）
- 当たった本体以外の円内敵に上記ダメージ
- 金色の円リング演出

### コイン

- 画面上の同時数に上限あり（60）。上限時は古いコインを1枚消してから新規ドロップ（ドロップ自体は止まらない）

### ターゲット維持（実装済み）

射程内の敵をロックし、射程外になるか撃破されるまで狙い続ける。

### XP 閾値（累計）

Lv2=4, Lv3=9, Lv4=15, Lv5=22…（次に必要な量は 4, 5, 6, 7… と増える）
